架設Hexo博客

關鍵詞

node.js, hexo, EPERM, github

注意事項

不要在Windows上安裝node.js或hexo，要在Linux上安裝，否則將遇到目錄層次太深導致EPERM錯誤問題，白白浪費時間。

github分支的名稱前綴要和用戶名完全一致，一個字符都不能變。比如我的用戶名是delphisfang，創建的分支就應該是delphisfang.github.io，而不能是delphis.github.io。

_config.yml中的冒號後面需要加空格。
